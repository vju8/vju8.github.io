Project 1: Fan Token Price Influence Analysis

This project analyzes the relationship between fan token prices and factors like match results, cryptocurrency indices (BTN, CIX100, CHZ) price fluctuations, and club transfers from the 18/19 to 22/23 football seasons. Using Selenium for data collection and scikit-learn for linear regression modeling, the study calculates expected and abnormal returns for fan tokens and benchmarks, evaluating the influence of various control variables such as match outcomes and venue.

Project 13:
Zombie Apocalypse Simulation

This Python project models a zombie apocalypse using mathematical principles inspired by Munz et al. (2009). The simulation involves humans, zombies, and deceased individuals, represented through a system of first-order Ordinary Differential Equations (ODEs). Users can explore the dynamics of the apocalypse under various parameters and scenarios.
